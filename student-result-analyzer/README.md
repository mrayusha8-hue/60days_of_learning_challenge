# Student Result Analyzer CLI

A Python-based command-line application to manage student records and analyze results.

## Features

- Add student records
- View all students
- Search student
- Update student details
- Delete student records
- Calculate percentage and grades
- Find topper
- Display class statistics
- Save and load data using JSON

## Technologies Used

- Python
- JSON

## Working
